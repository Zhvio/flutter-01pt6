# 📺 Flutter with Code

A simple Project with **FLutter**.

## ✨ Features
- **Login:** Login to your Account.
- **Register:** Register to your Account.
- **Online Database Tester:** Testing Upload Data with tester API.
- **View Data:** Viewing tester API data.
- **Changing and Removing data:** Editing API data.

## 🛠️ Requirements
Before running the app, ensure you have:
1. **IDE**
2. **Flutter** 
3. **Requirement PubSec**
    - flutter_easyloading: ^3.0.5
    - http: ^1.6.0
	
## 🚀 Installation

1. **Clone the repository:**
   ```Bash
   
   git clone https://github.com/Zhvio/flutter-01pt6
   ```
  
2. **Run It**

	```Bash
    Run with F5 (Debugging)
    ```	
	
3. **Usage** :

    a. Run the application:
    ```Bash
    F5 (debugging)
    ```
    b. Login User.
    
    c. Check View Upload or Update.
